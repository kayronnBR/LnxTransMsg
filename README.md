# LnxTrans

**LnxTrans** é um tradutor de texto otimizado para o protocolo **LnxTrans**, utilizando o poder do **Dialeto** para auxiliar na tradução de mensagens em jogos de forma rápida e fluida. (codigo feito pelo claude code)

-----

## 🚀 Como Configurar

### 1\. Download

Clique no botão abaixo para baixar os arquivos necessários:

<h1>
  <a href="https://downgit.github.io/#/home?url=https://github.com/kayronnBR/LnxTrans/blob/main/LnxTrans.sh">DOWNLOAD</a>
</h1>

### 2\. Preparação do Arquivo

1.  Extraia o conteúdo do arquivo baixado.
2.  Dê permissão de execução ao arquivo `LnxTrans.sh`. Você pode fazer isso via terminal com o comando:
    ```bash
    chmod +x LnxTrans.sh
    ```

### 3\. Obtendo o Caminho do Script

Para que o atalho funcione corretamente, você precisará do caminho completo do arquivo:

1.  Abra o seu terminal.
2.  Arraste o arquivo `LnxTrans.sh` para dentro da janela do terminal.
3.  O terminal exibirá o caminho completo (ex: `/home/usuario/Downloads/LnxTrans.sh`). **Copie este caminho.**

### 4\. Criando o Atalho de Teclado

Como o Wayland gerencia atalhos de forma específica, siga estes passos na sua interface (GNOME, KDE, etc.):

1.  Vá nas **Configurações do Sistema** \> **Teclado** \> **Atalhos de Teclado** (ou Atalhos Personalizados).
2.  Adicione um novo atalho.
3.  No campo **Nome**, digite: `LnxTrans`.
4.  No campo **Comando**, cole o caminho que você copiou no passo anterior.
5.  Escolha uma combinação de teclas de sua preferência (Exemplo: `CTRL` + `ENTER`).

-----

## 🕹️ Como Funciona

O uso do LnxTrans é simples e focado em produtividade durante a jogatina:

1.  **Ative o Script:** Pressione a tecla de atalho que você configurou (ex: `CTRL` + `ENTER`).
2.  **Digite sua Mensagem:** Uma caixa de texto aparecerá. Digite o que deseja traduzir e pressione `ENTER`.
3.  **Aguarde a Tradução:** O sistema emitirá um **aviso sonoro** assim que a tradução estiver pronta.
4.  **Cole no Jogo:** Vá para o chat do seu jogo e pressione `CTRL` + `V` para colar a mensagem já traduzida.
